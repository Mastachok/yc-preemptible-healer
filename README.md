# yc-preemptible-healer

Автоматически поднимает VM в Yandex Cloud, если они были остановлены.

## Возможности
- OAuth авторизация по ссылке
- Выбор VM по ID или имени
- systemd timer
- Цветное меню управления

## Установка
```bash
git clone https://github.com/Mastachok/yc-preemptible-healer
cd yc-preemptible-healer
bash install.sh
